Inspect the URL for this page and note the text after the question mark. The question mark and the text that follow is a "query string" sent from a form using method="GET".

?firstName=Fred&lastName=Jones

The first value in the query string starts with a "?" followed by a name and a value.

?firstName=Fred

The second and all remaining values in the query string starts with an "&" followed by a name and a value.

&lastName=Jones
